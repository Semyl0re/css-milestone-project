About the Project
This is my second milestone project for the TS Academy Frontend Development programme. The goal was to take a provided Figma mockup and recreate it as a fully styled, responsive webpage using HTML and CSS.
The page is themed around our solar system and presents planetary data in a clean, visual way. I built everything from scratch — structuring the HTML, writing an external CSS stylesheet, and making sure it works across different screen sizes.
What the Page Includes
 ∙ A navigation bar with a planet logo
 ∙ A hero section with a heading, description, and two call-to-action buttons
 ∙ An embedded video section about how planetary data helps us understand space
 ∙ An image grid showing visual comparisons of planets
 ∙ A data table with planetary facts taken from NASA
 ∙ A contact form with input fields, a city dropdown, radio buttons, and checkboxes
 ∙ A footer with copyright information and links.
